sscanf()、strcmp()、sort()、qsort()整理

2014/11/26 posted in  数据结构&算法

1.sscanf（）函数

作用：控制输入的格式

示例：sscanf(xs,"%s%s",trie[k].name1,trie[k].name2); //将数组xs内的字符串赋值给trie[k].name1，再输入一个字符串赋值给trie[k].name

具体用法：点击之前写过的博客: http://blog.csdn.net/codeball/article/details/39858145

2.strcmp（）函数

作用：比较两个字符串的大小，如果相等，返回0，如果前面大于后面（按字典序排列，越在前的字母，越小），返回1，否则返回-1.

示例：strcmp(sz1,sz2);

3.sort()函数

作用：排序

使用方法：sort(begin,end)，默认从小到大排序，或者sort(begin,end,compare)，compare为自定义的比较函数

排序的数据类型不局限于整数，只要是给定了小于运算就可以

4.qsort（）函数

作用：排序

使用方法：qsort ( 数组名 ，元素个数，元素占用的空间(sizeof)，比较函数)

比较函数：比较函数是一个自己写的函数 遵循 int com(const void *a,const void *b) 的格式。

当a b关系为 > < = 时，分别返回正值 负值 零 （或者相反）。
使用a b 时要强制转换类型，从void * 转换回应有的类型后，进行操作。
数组下标从零开始,个数为N, 下标0-(n-1)。

（1）对int数组排序：

int num[100];
int cmp ( const void *a , const void *b )
{
    return *(int *)a - *(int *)b;
}
qsort(num,100,sizeof(num[0]),cmp);

（2）对char数组排序 同int型，将int换为char

（3）对double数组排序：（和int不同，特别注意）

double in[100];
int cmp( const void *a , const void *b )
{
    return *(double *)a > *(double *)b ? 1 : -1;
}
qsort(in,100,sizeof(in[0]),cmp);

（4）对字符串进行排序：

struct In
{
    int data;
    char str[100];
} s[100];
//按照结构体中字符串str的字典顺序排序
int cmp ( const void *a , const void *b )
{
    return strcmp( (*(In *)a)->str , (*(In *)b)->str );
}
qsort(s,100,sizeof(s[0]),cmp);
