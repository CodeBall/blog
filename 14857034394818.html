Python strip()函数介绍

2015/10/28 posted in  Python

方法功能

strip()方法用于移除字符串头尾指定的字符

函数原型

str.strip(ch) 删除str字符串中开头,结尾处,位于ch删除序列的字符

str.lstrip(ch) 删除str字符串中开头处,位于ch删除序列的字符

str.rstrip(ch) 删除str字符串中结尾处,位于ch删除序列的字符

返回值

返回移除过之后的字符串

示例

str = "000this is a string\texample,let's look if carefuly!0000"
print "注意:只有两端的才有效果,中间的制表符是没有效果的"
print str.strip('0')
print str.lstrip('0')
print str.rstrip('0')

该段代码运行结果为:
this is a string example,let's look if carefuly!
this is a string example,let's look if carefuly!0000
000this is a string example,let's look if carefuly!
注意:

1.如果ch为空的话,默认删除空白符（包括'\n', '\r', '\t', ' ')

a = ' 123'
b = '\tabcd\t'
print a.strip()
print b.strip()

该段代码的运行结果为:
123
abcd

可见之前的空格和制表符的效果都消失了
2.这里的rm删除序列是只要边（开头或结尾）上的字符在删除序列内，就删除掉,不管顺序是怎么样的

c = "123abc"
print c.strip('21')
print c.strip('12')

这段代码的运行结果为:
3abc
3abc

可见两次处理结果是一样的
