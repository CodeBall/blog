include和require

这经常用在多文件开发中，一个文件不可能写完全部的代码，这会没有条理，看起来很糟糕，所以我们经常采用多文件的形式，将不同的功能放进不同的文件中，当在一个文件中调用另一个文件中的方法时，就要用include或者require去包含所用到的文件，才可以正常的引用。

两者的区别：

使用include，如果要找的文件不存在的时候，提示警告，程序可以正常运行

使用require，如果要找的文件不存在的时候，程序会提示一个错误，不能够继续运行下去。

以上两个当两次以上引用同一个文件的时候，就会产生重新定义函数的错误（如果没有函数，则不会报错，因为我们都知道，同一个函数名不能够重复定义，而每执行一次require或者include的时候，就相当于定义了一次函数）。

使用include_once和include_once就能够避免上述错误的产生。

PHP与MySQL（上篇）

MySQL一个库可以用来存放很多数据表。

创建一个库

打开浏览器，在地址栏中输入“localhost”——》点击进入“phpMyAdmin”——》就可以在左边看到很多文件夹，比如‘mysql’,'cdcol'，‘test’等，这些就是不同的库，现在，我们新建一个库——》点击菜单栏里面的数据库，填写数据库的名字和编码方式，一般会使用国际通用utf8——》点击创建，就可以成功地创建一个库了。

在库中添加一个数据表

单击点开要添加数据表的数据库——》新建数据表——》填写数据表的名字和字段数（即在这个表中存放几种数据，比如要存放ID，姓名，密码，年龄就是用了4个字段）——》点击执行（go）就成功创建了一个表——》之后就是向数据表中添加要存放的数据名称（ID，姓名，密码，年龄）——》点击保存（save）就完成了。

注意：（1）A_I指autoincreasment自行增加，可以用于ID等数据的输入。

（2）填写姓名可以写成是text类型，但是因为字符串很长（一般用于写博客的文本），所以可以选择是VARCHAR，自定义一个长度的字符.

（3）年龄的选择因为比较小，所以可以不必选择INT类型，可以用TINYINT

欧拉回路的应用

上一篇文章主要总结的是欧拉函数的基础知识，重点是欧拉回路的基本原理和判定是否能够组成欧拉回路，这篇文章重点总结欧拉回路在能够组成的基础上，输出路径问题。

查找欧拉回路的路径主要有两种方法，一种是DFS搜索，一种是Fleury（弗罗莱）算法。

DFS搜索

利用欧拉定理判断出一个图能够构成欧拉回路后，利用DFS算法遍历所有的边，并且保证每条边只遍历一遍。

用示例说明DFS算法：

题目描述：

给定n张骨牌，每张骨牌有左右两个点数（从1到6），问能不能交换骨牌的顺序交换左右两个点数，使得任意相邻骨牌的相邻字段为相等的数字。

输入描述：

输入文件中有多个测试数据。每个测试数据的第一行为一个整数N，1<=N<=100，表示多米诺骨牌的总数；接下来N行对每一张牌进行描述：每一行表示一张牌的左右两点的数字，用空格隔开。

输出描述：

对输入文件中的每个测试数据，如果不能满足要求，输出“No solution”；如果有解，输出任意一个即可：每张牌必须从左到右打印；每一行都要包含当前牌的序号和符号（+号代表没有翻转，-号代表左右翻转），每个测试数据的输出之后输入一个空行。

解析：这是一个无向图，骨牌可以翻转，而且一共有六个顶点，两个顶点之间可能有多条边，所以选择用邻接表存储图，每条边有两个方向，一正一负，这是需要记录的，每条边的两个顶点，边的序号都是需要记录的。通过边的度数判断能否构成欧拉通路，之后再利用DFS的搜索方法进行路径搜索。

存图——>判断能否构成欧拉回路——>DFS搜索路径——>打印路径

示例代码：

define MAXN 101

using namespace std;

struct EdgeNode
{
int adjvex;//边的令一个邻接点的序号
int EdgeNo;//边的序号
int flag;//判断边有没有反转，没有为1，翻转为-1
EdgeNode *next;//指向下一个边结点
};
EdgeNode *EdgeLink[10];//邻接表的头结点
int visited[MAXN];//记录每条边是否被遍历过
int path[MAXN];//记录加进欧拉图里的边，用边的序号标记
int pi;//记录当前已经加图里的边的数目
int N;//总边数
int du[10];//记录每个顶点的数目

void createLG()
{
int i,j,k,a,b;
EdgeNode p,q;
for(i = 1;i <= N;i++)
{
scanf("%d%d",&a,&b);
du[a]++;//整理度数
du[b]++;
p = new EdgeNode;//开辟第一个节点的内存
q = new EdgeNode;//开辟第二个节点的内存
p->adjvex = b;
p->EdgeNo = i;
p->flag = 1;
p->next = EdgeLink[a];
EdgeLink[a] = p;
q->adjvex = a;
q->EdgeNo = i;
q->flag = -1;
q->next = EdgeLink[b];
EdgeLink[b] = q;
}
}

void DFS(int start)
{
while(pi <= N)
{
EdgeNode *p;
p = EdgeLink[start];
while(p != NULL)
{
if(!visited[p->EdgeNo])
{
//标记改边已经被遍历过了
visited[p->EdgeNo] = 1;
//根据判断该边是否翻转决定存入数组数据正负号
if(p->flag > 0)
path[pi] = p->EdgeNo;
else
path[pi] = -p->EdgeNo;
pi++;
DFS(p->adjvex);
}
else
p = p->next;
}
}
}

void Domino()
{
int start1,start2,jd = 0;
//判断能不能构成欧拉路
for(int i = 1;i <= 6;i++)
{
if(du[i] % 2)
{
start1 = i;
jd++;
}
if(du[i] != 0)
start2 = i;
}
if(jd != 0 && jd != 2)
{
printf("NO Solution!\n");
return ;
}
pi = 1;
if(jd == 2)
DFS(start1);//有奇度的点，从奇度的点开始出发
else
DFS(start2);//没有奇度的点，从任一有边的点出发都可以
char flag1 = '+',flag2 = '-';
for(int i = 1;i < pi;i++)
{
if(path[i] > 0)
printf("%d%c\n",path[i],flag1);
else
printf("%d%c\n",-path[i],flag2);
}
}

void delet()
{
EdgeNode *p;
for(int i = 1;i <= 6;i++)
{
p = EdgeLink[i];
while(p != NULL)
{
EdgeLink[i] = p->next;
delete p;
p = EdgeLink[i];
}
}
}

int main()
{
while(~scanf("%d",&N))
{
memset(du,0,sizeof(du));
memset(visited,0,sizeof(visited));
memset(path,0,sizeof(path));
for(int i = 1;i <= 6;i++)
EdgeLink[i] = NULL;
createLG();//建图，记录度数
Domino();
delet();//用来释放内存
}
return 0;
}
