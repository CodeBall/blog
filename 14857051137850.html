连接数据库

对于用户的各种操作，都离不开数据库，所以在操作之前就要连接好数据库

//定义几个常量
define('MYSQL_HOST','localhost');
define('MYSQL_USER','root');
define('MYSQL_PW','');
function connectDb()
{
\(conn = mysql_connect(MYSQL_HOST,MYSQL_USER,MYSQL_PW);
mysql_select_db('myapp');
return \)conn;
}

删除数据

删除数据比较简单，只需要根据要删除的数据的id进行删除操作即可

if(!empty(\(_GET['id']))
{
\)conn = connectDb();
\(id = \)_GET['id'];

}
